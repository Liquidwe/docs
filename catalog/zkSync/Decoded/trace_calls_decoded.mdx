### Table Description
The zksync.trace_calls_decoded table provides a comprehensive and human-readable view of the internal transactions and function calls within the Ethereum blockchain. By decoding trace calls, this table enables easier understanding and analysis of the complex interactions and operations that occur during transaction execution.
